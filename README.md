
# DoItNow - Simple Todolist


DoItNow is a dynamic web application built on EJS and Express.js to implement the aforementioned technologies.

The app asks the user to easily register by simply providing a name, then they are redirected to one of the sections (today & work) where the user can start adding tasks to be done, which can be marked as completed.

## Tech Stack

**Client:** EJS, CSS

**Server:** Node, Express
